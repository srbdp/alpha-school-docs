# Alpha School Documentation

Welcome to the official Alpha School documentation. This resource provides structured, comprehensive information about Alpha School's innovative educational model, curriculum, enrollment process, and outcomes.

## What is Alpha School?

Alpha School is a revolutionary K-12 private school that combines AI-powered personalized learning with human mentorship. Students complete core academics in just 2 hours per day, spending the remaining time on passion projects, physical activity, and real-world skill development.
